# Angular Products App — feature/api

Rama dedicada a la integracion del backend REST API con Spring Boot.

## Estado

En construccion. Esta rama conectara el frontend Angular con la API de Spring Boot.

## Lo que se integrara

- Consumo de endpoints REST con HttpClient
- Servicios Angular para productos
- Spring Boot + Spring Data JPA + MySQL

## Rama base

Esta rama se deriva de `feature/form` que contiene el formulario con validaciones.
