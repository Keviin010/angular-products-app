# Angular Products App — feature/form

Rama que contiene el formulario completo de gestion de productos.

## Funcionalidades en esta rama

- Formulario para agregar productos
- Formulario para editar productos
- Eliminacion de productos
- Validaciones en todos los campos
- Notificaciones toast de acciones exitosas

## Servidor de desarrollo
```bash
ng serve
```

Abre el navegador en `http://localhost:4200/`.

## Rama base

Esta rama se deriva de `main` que contiene el listado de productos.
